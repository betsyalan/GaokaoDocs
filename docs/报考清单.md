# 2026年广东高考志愿填报准备任务清单

> 🎯 核心原则：考完前专心考试，考完后按部就班准备，**排位比分数更重要**。

---

## 🔴 第一阶段：考前及考后立即行动（6月3日 — 6月9日）

### 👤 考生任务
- [ ] 保持正常作息，调整心态，回归基础，不再钻研难题
- [ ] 6月9日考完最后一科后，**立即对照官方答案进行估分**，记录各科预估分和总分
- [ ] 将估分结果告知家长，用于后续初步定位

### 👪 家长任务（可同步悄悄进行）
- [ ] 访问 **广东省教育考试院官网**（eea.gd.gov.cn），通读《广东省2026年普通高等学校招生工作规定》
- [ ] 弄懂核心规则：
  - [ ] “院校专业组” 平行志愿投档方式（分数优先、遵循志愿、一次投档）
  - [ ] 各批次志愿数量（如本科批45个院校专业组，每组6个专业）
  - [ ] 填报时间节点和流程
- [ ] 下载往年数据：
  - [ ] 从考试院官网获取2025、2024年各高校在粤投档最低分及**最低排位**
  - [ ] 保存好这些排位数据，建立初步参照表
- [ ] 准备官方指导材料：
  - [ ] 留意学校下发《广东省2026年普通高等学校招生专业目录》和《志愿填报指南》的时间，确保能第一时间拿到

---

## 🔵 第二阶段：出分前黄金准备期（6月10日 — 出分当日）

### 🔍 自我认知与专业方向探索
- [ ] 完成兴趣与能力梳理：
  - [ ] 问自己：喜欢与人打交道，还是与数据/机械/创意打交道？
  - [ ] 分析擅长能力：数理逻辑、语言表达、动手操作、空间想象、共情沟通等
  - [ ] 想清楚未来生活倾向：高薪高压、稳定规律、自由创造、专业精深？
- [ ] 完成 **霍兰德职业兴趣测试**（仅作参考，不迷信结果）
- [ ] 结合以上结果，初步圈定 2-3 个不排斥的学科门类（如计算机/电子、医学、经管、法学、师范、公安等）

### 📋 锁定专业与院校范围
- [ ] 用估分换算预估排位区间（上下浮动2000名左右）
- [ ] 查找并核对选考科目要求：
  - [ ] 在广东省教育考试院官网搜索下载 **《2026年拟在粤招生普通高校本科招生专业选考科目要求》** Excel表格
  - [ ] 使用筛选功能，逐一核对心仪专业是否与你的“3+1+2”选科组合匹配
- [ ] 检查自身高考体检情况，对照《招生专业目录》标注的受限专业，排除因身体条件不能报考的方向
- [ ] 对照2025、2024年投档排位，找出排位在预估区间内的所有“院校专业组”，整理成**院校池**长名单
- [ ] 按“冲、稳、保”策略分组：
  - [ ] 冲：20%专业组（排位略高于或持平于你的预估最高排位）
  - [ ] 稳：50%专业组（排位稳定低于预估排位，录取概率大）
  - [ ] 保：30%个专业组（排位远低于预估排位，确保录取）

### 🔬 深度研究每一个目标专业组
- [ ] 去目标高校官网招生专栏，逐份阅读《招生章程》
- [ ] 重点记录：
  - [ ] 专业录取规则（分数优先、专业志愿优先、是否有专业级差）
  - [ ] 是否有单科成绩、外语语种、性别等特殊要求
  - [ ] 校区地址、学费标准
- [ ] 分析每个专业组内的专业构成：
  - [ ] 列出组内所有专业，标记哪些是你能接受的，哪些是难以接受的
  - [ ] 评估被调剂到冷门专业的风险承受度，若完全不能接受，则降低该专业组的填报优先级

---

## 🟢 第三阶段：出分后至志愿填报截止

### ✏️ 精准修正与决策
- [ ] 用真实高考成绩和全省排位，替换之前的预估排位
- [ ] 对“院校池”和“冲稳保”分组进行快速修正，剔除已明显不合要求的选择
- [ ] 确定最终45个院校专业组的排列顺序，按录取意愿从高到低排列
- [ ] 每个专业组内，再次核对6个专业志愿的顺序，将最想读的专业放前面

### 💻 系统操作与正式提交
- [ ] 在广东省教育考试院志愿填报系统开放期间，进行模拟填报，熟悉操作流程
- [ ] 在官方截止时间前，正式完成网上填报
- [ ] 反复核对已填信息，特别是专业组代码、专业代码
- [ ] **务必完成网上确认操作**，确认提交成功，并截图或拍照保存凭证
