## 第三章 打破信息差：潜在短板与就读挑战

> **以下内容基于网络公开的在校生反馈、行业论坛讨论等整理，属口碑参考，非官方数据。**

### 3.1 课程陈旧与产业脱节问题

- **计算机/电子类**：核心课程设置较为传统（组成原理、操作系统、编译原理等），与产业前沿（大模型、AI工程化）的衔接主要依赖选修课和自主学习[reference:85]。但学校已通过增设“伏羲班”“腾创班”等创新班弥补这一不足[reference:86]。
- **机械类**：课程体系相对成熟，“3+1”模式注重实践[reference:87]。行业反馈整体较好。
- **总体评价**：普通班课程偏传统，创新班/特色班课程更前沿。

### 3.2 压分现象与导师“放养”情况

- **压分现象**：该数据暂未查到明确的在校生大规模反馈。
- **导师“放养”** ：该数据暂未查到系统性评价。不同导师风格差异较大，建议考研/保研学生提前了解目标导师课题组情况。

### 3.3 校区偏僻与实习通勤问题

- 广工主校区（大学城校区）位于广州市番禺区，地铁4号线/7号线可达，**通勤条件良好**。
- 揭阳校区为新校区，地理位置相对偏远[reference:88]，实习通勤可能存在不便，报考时需注意区分校区。

### 3.4 毕业设计与延毕率

- 该数据暂未查到关于延毕率的官方统计或大规模在校生反馈。
- 机械工程、自动化等工科专业毕业设计通常要求较高（含实物制作/系统实现），建议提前规划。

## 第五章 求职/升学时间线（大四/研三 9月→次年6月）

### 本科生（大四）求职时间线

| 时间节点 | 事项 | 建议 |
|---------|------|------|
| **9月** | 秋招启动、互联网/科技企业提前批 | 完成简历打磨，关注华为、腾讯、大疆等校招官网 |
| **10月** | 秋招黄金期、各大企业笔试面试 | 刷LeetCode/行测，参加校内宣讲会 |
| **11月** | 秋招高峰期、部分企业发放Offer | 南方电网等国企启动招聘（电气专业重点关注）[reference:102] |
| **12月** | 秋招收尾、考研初试 | 已拿Offer的同学可冲刺考研；未拿Offer的准备春招 |
| **1月** | 寒假、考研初试后调整 | 复盘秋招，补充项目/实习经历 |
| **2-3月** | 春招启动、考研成绩公布 | 成绩理想的准备复试；不理想的全力春招 |
| **4月** | 春招高峰期、公务员/事业单位考试 | 关注各类招考信息 |
| **5月** | 春招收尾、毕业设计答辩 | 确保毕业设计通过，确认入职Offer |
| **6月** | 毕业、入职准备 | 办理入职手续，完成从学生到职场人的转变 |

| 时间节点 | 事项 | 建议 |
