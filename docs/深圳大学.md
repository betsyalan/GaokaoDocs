# 深圳大学工科深度调研报告

**聚焦学科**：计算机科学与技术 / 电子科学与技术 / 电气工程 / 机械工程

**报告说明**：本报告基于教育部学科评估、ESI排名、QS世界大学学科排名、深圳大学官方就业质量报告、阳光高考信息平台等公开数据，以及网络论坛在校生反馈等口碑信息编制而成。所有数据均已注明来源，未能查实的数据已标注“暂未查到”。


## 执行摘要（SWOT 快速决策卡）

### 优势（Strengths）
- **地理位置无可替代**：校园周边3公里范围内云集腾讯、华为、中兴、大疆等200余家上市公司国际总部[reference:0]
- **计算机学科实力强劲**：ESI全球前0.454‰（全球38位、全国第16），已通过国际工程教育专业认证[reference:1]
- **就业质量突出**：近50%毕业生入职世界500强，2023-2025年录用人数最多企业为比亚迪、华为、新凯来、腾讯[reference:2]
- **科研平台高端**：拥有全国重点实验室5个（1个牵头、4个共建），包括射频异质异构集成全国重点实验室[reference:3][reference:4]

### 劣势（Weaknesses）
- **保研率偏低**：全校整体仅6.14%，远低于985高校平均水平[reference:5]
- **学科评估不均衡**：电子科学与技术（第四轮C-）、机械工程（第四轮C-）实力较弱[reference:6]
- **“双非”身份限制**：部分头部企业核心岗位（如腾讯算法岗）倾向“双9”或C9硕士背景[reference:7]
- **电气工程已停招**：该专业因就业落实率不达标已被列入停招名单[reference:8]

### 机会（Opportunities）
- **深圳“20+8”产业集群**：半导体、人工智能、智能网联汽车等8个集群实现两位数增长[reference:9]
- **集成电路学院新成立**：依托全国重点实验室，构建“本-硕-博”贯通培养体系[reference:10]
- **大湾区人才需求旺盛**：深大毕业生留深率72.68%，大湾区就业比例85.53%[reference:11]
- **特色班保研率极高**：卓越班推免率可超30%，计算机卓越班达29.41%[reference:12][reference:13]

### 威胁（Threats）
- **课程体系争议**：部分在校生反馈课程设计不够体系化、部分教材陈旧[reference:14][reference:15]
- **导师质量参差**：存在“放养”型导师和个别导师压榨学生的口碑[reference:16][reference:17]
- **传统工科薪资天花板**：机械等传统工科起薪显著低于计算机、电子[reference:18]
- **双非学历在部分领域受限**：选调生、央企总部等岗位门槛可能受限

## 第三章 打破信息差：潜在短板与就读挑战

### 3.1 课程陈旧与产业脱节问题

- **计算机专业**：部分在校生反馈“课程不够精，很多就自己辅修”，计算机“大部分东西还是得靠自学”[reference:96][reference:97]。新成立的IEEE荣誉班“课程体系还在磨合，首批毕业生去向未知”[reference:98][reference:99]。
- **电子专业**：课程难度大、挂科率高[reference:100]。部分课程存在“重点难点不突出，教师讲解不到位”的问题[reference:101]。有校友反馈“本科课程水，但是师资和实验室配套强大”[reference:102]。
- **整体评价**：有毕业生对比末流985后指出，深大“课程设计不是很体系，本科生无法通过课程学习比较全面地了解和熟悉学科，导致基础不是很扎实”[reference:103]。部分专业存在“教材陈旧”的问题[reference:104]。

### 3.2 压分现象与导师“放养”情况

**压分问题**：
- 考研界传闻深圳大学存在压分现象[reference:105]。有个别导师被指“压低其专业课的分数”[reference:106]。
- 注：此为网络口碑，非官方确认数据。

**导师“放养”问题**：
- 有导师明确表示“培养方式以散养为主”[reference:107]。
- 深大遵循导师双选模式，部分导师“事业上功成名就，生活里异常忙碌。容易形成散养式教学，一学期不知道能见几次”[reference:108]。
- 2022年曾发生研究生实名举报导师压迫事件，校方已介入处理[reference:109][reference:110]。

### 3.3 校区与实习通勤

- **校区位置**：粤海校区位于深圳市南山区南海大道3688号，丽湖校区位于南山区学苑大道1066号[reference:111]
- **实习便利性**：**极大优势**。校园周边3公里范围内云集腾讯、华为、中兴、大疆等200余家上市公司[reference:112]。在深大就读“实习方便，周围中大厂挺多的”[reference:113]。
- **结论**：不存在校区偏僻影响实习的问题，反而是全国实习最便利的高校之一。
