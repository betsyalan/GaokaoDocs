# 高考录取数据集成 — 设计文档

日期：2026-06-10

## 概述

在现有 GaokaoDocs 文档内容管理系统中，集成 `data/gaokao_scores.db` 数据库中的高考录取数据。在保持现有三套主题、侧栏分类体系和交互模式不变的前提下，新增「历年录取分」和「一分一段表」两个数据分类，用于浏览和查询录取分数数据。

## 数据源

数据库 `data/gaokao_scores.db`，涉及以下表：

| 表名 | 行数 | 用途 |
|---|---|---|
| `university_admission_data` | 1442 | 大学录取详情（专业级数据），含分数线、排位 |
| `universities_info` | 19 | 大学基本信息（所在地省份、城市、985/211/双一流标记） |
| `province_score_distribution` | 597 | 广东省 2025 物理类一分一段表 |

## 侧栏结构

在现有侧栏分类末尾新增两个分类，与现有「专业信息」「大学信息」等分类并列，共享相同的展开/收起交互：

```
▸ 专业信息
▸ 大学信息
▸ 报考指南
▸ 志愿数据
▸ 参考资料
▸ 其他页面
▸ 历年录取分              ← 新增
  ▸ 广东省 (6)             ← 按大学所在地省份分组，可收缩
     ├ 中山大学
     ├ 华南理工大学
     ├ ...
  ▸ 江苏省 (4)
  ▸ 北京市 (4)
  ▸ 陕西省 (2)
  ▸ 湖北省 (1)
  ▸ 安徽省 (1)
  ▸ 四川省 (1)
▸ 一分一段表              ← 新增（叶子节点，直接点击展示数据）
```

**交互规则：**
- 省份组可折叠展开（与现有分类展开/收起行为一致）
- 点击大学名称 → 右侧展示该大学的录取数据详情
- 点击「一分一段表」→ 右侧展示一分一段数据表格

## 配色方案

新分类的徽标色使用 `--accent-color` CSS 变量驱动，与三套主题联动：

| 主题 | `--accent-color` | 历年录取分徽标色 | 一分一段表徽标色 |
|---|---|---|---|
| 琉璃蓝 | `#1e6bb8` | `#1e6bb8` | `#1e6bb8` |
| 琥珀橙 | `#c94f2b` | `#c94f2b` | `#c94f2b` |
| 紫烟 | `#7c4d9e` | `#7c4d9e` | `#7c4d9e` |

图标从 Lucide Icons（`lucide-vue-next`）选取，与项目现有图标风格一致：
- 历年录取分：`ScrollText` 📜
- 一分一段表：`BarChart4` 📊
- 省份组展开箭头：复用现有 `▾` 指示器

## 新增 API 接口

| 方法 | 路径 | 返回 |
|---|---|---|
| GET | `/api/gaokao/universities` | 大学列表，按所在地省份分组，含标签信息（985/211/双一流等） |
| GET | `/api/gaokao/admission/:code` | 指定大学（按 code）的录取数据，按专业组分组，含专业明细 |
| GET | `/api/gaokao/distribution` | 一分一段表全部数据（当前仅广东 2025 物理类） |

## 页面设计

### 一分一段表页面

点击侧栏「一分一段表」，右侧展示数据表格。无上部 Tab、无筛选栏（数据量小，仅广东 2025）：

```
📊 一分一段表              ← .page-title 风格

分数   本段人数  本科累计  专科累计
697     19        19        19
696     13        32        32
695      7        39        39
...                      ← 共 597 行

共 597 条记录（广东省 · 2025 · 物理类）
```

点击侧栏中某所大学，右侧展示该大学录取数据：

```

▾ 元班专业组                     ← 可折叠的专业组标题

   电子信息类(元班)    4   658  657  658
```

## UI 样式规则

所有数据展示组件遵循现有 CSS 变量体系：

| 样式属性 | 使用的 CSS 变量 |
|---|---|
| 容器背景 | `var(--card-bg)` |
| 容器阴影 | `var(--card-shadow)` |
| 容器圆角 | `8px`（与 FileCard 一致） |
| 边框 | `var(--border-color)` |
| 标题色 | `var(--text-primary)` |
| 副文本色 | `var(--text-secondary)` |
| 表头背景 | `var(--body-bg)` |
| 行 hover | 浅色高亮，适配各主题 |
| 页码/副信息 | `var(--text-secondary)` |

## 路由设计

复用 Vue Router 已有模式，新增两个路由：

```js
{ path: '/gaokao/distribution', name: 'GaokaoDistribution', component: ... }
{ path: '/gaokao/admission/:code', name: 'GaokaoAdmissionDetail', component: ... }
```

两个新页面均为独立组件，放入 `frontend/src/views/` 目录。

## 实施范围与限制

- 当前数据仅覆盖广东省 2025 年物理类，但架构支持未来扩展其他省份和年份
- 侧栏大学列表按所在地省份分组，数据来源于 `universities_info` 表
- 录取数据展示只读，不支持编辑
- 不涉及文件上传/管理功能
